Deskripsi Data :

Data ini berisi tentang mencapai target pelelangan, apakah penawar memenangkan product dengan harga tersebut atau tidak (di tempat lelang).Kenapa data ini bisa disebut penting untuk para penawar? karna dari hasil klasifikasi ini para penawar bisa memperkirakan hal-hal yang perlu disiapkan untuk mencapai target pelelangan.

fitur ini berisi tentang penawaran sebuah product dari penawar 1- penawar 4,yang berisi 2043 data. dengan parameter kemenangan : harga,produk dan berapa kali penawar memenangkan pelelangan.

tujuan dari data ini adalah para penawar bisa memperkirakaan kemenangan pelelangan dengan parameter kemenangan dilihat dari harga,produk dan berapa kali penawar memenangkan pelelangan.

Data ini memverifikasi apakah harga tertentu mungkin untuk produk tertentu, dan apakah pemenang lelang tertentu kemungkinan besar memenangkan produk pada harga tersebut.

Terdapat 7 fitur yaitu :

  1. process.b1.capacity (penawar 1)

  2. process.b2.capacity (penawar 2)

  3. process.b3.capacity (penawar 3)

  4. process.b4.capacity (penawar 4)

  5. property.price (harga)

  6. property.product (produk)

  7. property.winner (pemenang)

Dengan label target

  • verification.result (True and False)

Penjelasan Model dan Preprocecing

MLP

Multilayer Perceptron (MLP) adalah salah satu jenis arsitektur dalam jaringan saraf tiruan (Neural Network). MLPClassifier adalah implementasi dari MLP untuk tugas klasifikasi dalam library scikit-learn. Arsitektur umum dari MLP dapat dijelaskan dengan beberapa komponen utama. Berikut adalah rumus-rumus dasar yang terkait dengan MLPClassifier: MLP terdiri dari input layer, satu atau lebih hidden layer, dan output layer. Berikut penjelasan masing-masing layer:

  • Input layer Input layer untuk menerima nilai masukan dari tiap record pada data. Jumlah simpul input sama dengan jumlah variabel prediktor.

  • Hidden layer Hidden layer mentransformasikan nilai input di dalam network. Tiap simpul pada hidden layer terhubung dengan simpulsimpul pada hiden layer sebelumnya atau dari simpul-simpul pada input layer dan ke simpul-simpul pada hidden layer berikutnya atau ke simpul-simpul pada output layer. Jumlah hidden layer bisa berapa saja.

  • Output layer Garis yang terhubung dengan Output layer berasal dari hidden layer atau input layer dan mengembalikan nilai keluaran yang bersesuaian dengan variabel prediksi. Keluaran dari output layer biasanya merupakan nilai floating antara 0 sampai 1

import library

  • numpy (diimport sebagai np): Library untuk operasi numerik efisien.

  • import pandas as pd: Mengimpor perpustakaan Pandas dengan alias pd. Pandas digunakan untuk manipulasi dan analisis data, dan sering digunakan untuk membaca dataset.

  • train_test_split dari sklearn.model_selection: Digunakan untuk membagi dataset menjadi dua bagian, yaitu data latih (training data) dan data uji (test data).

  • PCA dari sklearn.decomposition: Digunakan untuk analisis komponen utama (Principal Component Analysis).PCA digunakan untuk mereduksi dimensi data dengan mempertahankan sebanyak mungkin informasi. Ini dapat membantu mempercepat pelatihan model dan mengurangi overfitting.

  • StandardScaler digunakan untuk melakukan penskalaan fitur, memastikan bahwa fitur memiliki rata-rata nol dan deviasi standar satu. Ini dapat membantu meningkatkan konvergensi model.

  • MLPClassifier dari sklearn.neural_network: Implementasi dari algoritma klasifikasi menggunakan jaringan saraf tiruan (Multilayer Perceptron).

  • pickle digunakan untuk menyimpan model atau objek Python ke dalam file, dan kemudian membacanya kembali. Ini memungkinkan untuk menyimpan model setelah pelatihan dan menggunakannya kembali tanpa harus melatih ulang.

# Import library

‘data.csv’: Parameter pertama berisi nama file CSV yang akan dibaca.

membagi data menjadi training dan testing set

  • train_test_split(X, y, test_size=0.1, random_state=42):

Fungsi ini mengambil empat argumen:

  1. X: Fitur-fitur yang ingin dibagi (dalam hal ini, DataFrame X yang berisi fitur-fitur).

  2. y: Target atau label yang ingin dibagi (dalam hal ini, Series y yang berisi label).

  3. test_size: Proporsi data yang akan dialokasikan sebagai set pengujian. Dalam contoh ini, test_size=0.1 berarti 10% dari data akan menjadi set pengujian, sedangkan 90% akan menjadi set pelatihan.

  4. random_state: Digunakan untuk memastikan hasil yang reprodusibel. Angka apa pun dapat digunakan sebagai nilai ini.

  • X_train, X_test, y_train, y_test =

Variabel X_train dan y_train akan berisi set pelatihan untuk fitur dan target. Variabel X_test dan y_test akan berisi set pengujian untuk fitur dan target. Setelah menjalankan baris kode ini, kita memiliki empat variabel yang memuat data terbagi, yaitu:

X_train: Fitur dari set pelatihan. X_test: Fitur dari set pengujian. y_train: Target dari set pelatihan. y_test: Target dari set pengujian.

# Membagi data menjadi training dan testing set
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.1, random_state=42)

standarisasi fitur

StandardScaler digunakan untuk melakukan penskalaan fitur sehingga setiap fitur memiliki rata-rata nol dan deviasi standar satu. Proses ini berguna untuk mengatasi perbedaan skala antar fitur, yang dapat memengaruhi performa beberapa model pembelajaran mesin.

  • nnX_test = scaler.fit(X_test): Metode fit dari StandardScaler digunakan untuk menghitung rata-rata dan deviasi standar dari set pengujian (X_test). Hasilnya disimpan dalam variabel nnX_test.

Membangun dan melatih model MLPClassifier

MLPClassifier adalah model yang dapat digunakan untuk tugas klasifikasi dengan memanfaatkan kekuatan dari jaringan saraf tiruan.MLPClassifier (Multi-Layer Perceptron Classifier) adalah model klasifikasi yang termasuk dalam kategori jaringan saraf tiruan (neural networks) dan disediakan oleh pustaka scikit-learn di Python.

-(Hidden Layers): MLPClassifier dapat memiliki satu atau lebih lapisan tersembunyi. Jumlah dan ukuran neuron dalam setiap lapisan dapat dikontrol oleh parameter hidden_layer_sizes. hidden_layer_sizes=(100, 50, 25) Ada tiga lapisan tersembunyi di jaringan saraf:

  • Lapisan pertama memiliki 100 unit atau neuron.

  • Lapisan kedua memiliki 50 unit atau neuron.

  • Lapisan ketiga memiliki 25 unit atau neuron.

Setiap unit dalam lapisan tersembunyi berperan sebagai pembangun blok untuk pembelajaran pola yang lebih kompleks dari data pelatihan. Jumlah unit dalam lapisan tersembunyi adalah salah satu faktor yang dapat mempengaruhi kapasitas model, di mana model dengan lebih banyak unit mungkin dapat mempelajari pola yang lebih rumit, tetapi juga memerlukan lebih banyak data dan waktu pelatihan.

  • max_iter=1000: Ini menentukan jumlah iterasi maksimum atau epoch yang akan dilakukan selama pelatihan model. Dalam hal ini, pelatihan akan berhenti setelah 1000 iterasi jika konvergensi belum tercapai.

  • activation=’relu’: Ini menentukan fungsi aktivasi yang digunakan di setiap unit atau neuron. ‘relu’ mengacu pada fungsi aktivasi ReLU (Rectified Linear Unit), yang umum digunakan dalam lapisan tersembunyi.

  • random_state=42: Parameter ini memberikan nilai seed untuk menghasilkan hasil yang dapat direproduksi, seperti yang telah dijelaskan sebelumnya.

mlp.fit(X_train, y_train): Ini adalah langkah pelatihan model. Metode fit digunakan untuk melatih model dengan menggunakan data pelatihan.

  • X_train: Subset fitur untuk pelatihan model.

  • y_train: Subset target untuk pelatihan model.

Setelah pemanggilan fit, model akan belajar dari data pelatihan dan dapat digunakan untuk membuat prediksi pada data baru.

# Membangun dan melatih model MLPClassifier
mlp = MLPClassifier(hidden_layer_sizes=(100, 50, 25), max_iter=1000, activation='relu', random_state=42)
mlp.fit(X_train, y_train)

Evaluasi model pada data pengujian

  1. accuracy = mlp.score(X_test, y_test):

  • mlp: Merujuk ke objek model MLPClassifier yang telah dilatih sebelumnya.

  • score(…): Metode score digunakan untuk mengukur akurasi model pada dataset yang diberikan. Akurasi diukur sebagai rasio jumlah prediksi yang benar terhadap jumlah total data.

  • X_test: Subset fitur dari data pengujian.

  • y_test: Subset target dari data pengujian.

Hasil dari pemanggilan score disimpan dalam variabel accuracy.

  1. print(f’Accuracy: {accuracy * 100:.2f}%’):

  • f’Accuracy: {accuracy * 100:.2f}%’: Ini adalah f-string, yang digunakan untuk memasukkan nilai variabel (accuracy) ke dalam string. Angka setelah titik dua (:.2f) mengontrol jumlah digit desimal yang ditampilkan.

  • accuracy * 100: Mengubah nilai akurasi menjadi persentase.

  • :.2f: Menunjukkan dua digit desimal.

Hasil akhirnya adalah mencetak akurasi model pada dataset pengujian ke konsol dengan format “Accuracy: xx.xx%”.

# Evaluasi model pada data pengujian
accuracy = mlp.score(X_test, y_test)
print(f'Accuracy: {accuracy * 100:.2f}%')
